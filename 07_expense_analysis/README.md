# Задание 7 - Анализ затрат офиса

Программа для анализа расходов офиса по категориям.

## Что делает
- читает данные из CSV-файла
- считает расходы по категориям
- сравнивает расходы с бюджетом
- выводит общую статистику

## Формат CSV
```csv
category,amount,budget
Phone,1200,1500
Internet,2500,3000
Supplies,1800,1500
```

## Запуск
```sh
python3 expense_analysis.py expenses.csv
```
