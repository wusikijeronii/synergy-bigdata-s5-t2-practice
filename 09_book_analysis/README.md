# Задание 9 - Анализ электронной книги

Программа для анализа книги в текстовом формате.

## Что делает
- считает количество строк
- считает количество слов
- считает количество абзацев
- считает количество глав
- определяет примерный уровень чтения
- выводит условный показатель популярности

## Запуск

```bash
python3 book_analysis.py book.txt
```
