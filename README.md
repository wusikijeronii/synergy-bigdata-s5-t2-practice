# Big Data Practice (Synergy University, Semester 5)
## EN

This repository contains simplified implementations of practical tasks for the Big Data systems design course.

Each task demonstrates basic concepts of:

* databases
* data processing
* analytics
* system modeling

All implementations are educational and intentionally simplified.

## RU

Репозиторий содержит упрощённые реализации практических заданий по курсу проектирования систем обработки больших данных.

Каждое задание демонстрирует базовые подходы к:

* работе с базами данных
* обработке данных
* аналитике
* моделированию систем

Все реализации выполнены в учебных целях и упрощены.
