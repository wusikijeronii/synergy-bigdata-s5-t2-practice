# Задание 4 - Диагностика блока питания

Упрощённая программа на C++ для диагностики возможных проблем блока питания.

## Что делает
- принимает значения температуры
- принимает нагрузку
- принимает потребляемую мощность
- принимает напряжение и скорость вентилятора
- выводит предупреждения о возможных проблемах

## Сборка

```bash
g++ power_supply_diagnostics.cpp -o power_supply_diagnostics
```

## Примечание

Решение выполнено в учебных целях в упрощённом виде. Программа анализирует введённые данные и не получает информацию напрямую с оборудования.
