# Задание 6 — Приложение с информацией об автомобилях

Консольное приложение на Java для выбора автомобиля и вывода информации о нём.

## Что делает
- пользователь вводит марку, модель и год
- программа ищет автомобиль в базе
- выводит характеристики и среднюю стоимость

## Запуск

```bash
javac CarInfoApp.java
java CarInfoApp
```

## Примечание

Решение выполнено в учебных целях. Данные об автомобилях заданы внутри программы.
