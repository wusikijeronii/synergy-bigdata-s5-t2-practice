# Задание 10 - Симулятор производственного процесса на складе

Упрощённая программа для моделирования обработки заказов на складе.

## Что делает
- обрабатывает заказы
- учитывает остаток продукции
- считает затраты на материалы
- определяет количество бракованных изделий
- оценивает общее время сборки
- выводит простой вариант графика работы персонала

## Запуск

```bash
python3 warehouse_sim.py
```
